# Telegram-бот для nail-мастера

Бот на `aiogram 3` для записи клиентов на услуги мастера. Есть пользовательское меню, портфолио, отзывы, FAQ, контакты и Telegram-CRM для мастера внутри `/admin`.

## Возможности

- `/start` и главное меню для клиента.
- Запись на одну или несколько услуг за один визит.
- Автоматический расчет суммы, длительности и времени окончания.
- Проверка занятости по интервалам, а не только по одинаковому времени старта.
- Уведомления мастеру о новых записях.
- Админ-панель `/admin` с ролями и разными правами доступа.
- Управление услугами, ценами, записями, клиентами, отзывами и закрытыми датами прямо в Telegram.
- CSV-выгрузка записей для Excel.

## Быстрый запуск

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
Copy-Item .env.example .env
```

Заполните `.env`:

```dotenv
BOT_TOKEN=токен_от_BotFather
OWNER_ID=telegram_id_создателя
MASTER_CHAT_ID=telegram_id_мастера
ADMIN_IDS=старый_список_админов_для_совместимости
```

Запуск:

```powershell
python -m app.main
```

## Первичная настройка

1. Заполнить `BOT_TOKEN`.
2. Заполнить `OWNER_ID`.
3. Запустить бота.
4. Написать `/myid`, если ID неизвестен.
5. Владелец назначает мастера:

```text
/setrole 123456789 super_admin
```

6. Мастер управляет ботом через `/admin`.

`MASTER_CHAT_ID` и `ADMIN_IDS` оставлены для совместимости со старой конфигурацией. При запуске они импортируются как `super_admin`, если таких пользователей еще нет в списке ролей.

## Роли

- `owner` — создатель бота. Может все, включая назначение ролей. Задается через `OWNER_ID`.
- `super_admin` — мастер или владелица салона. Может управлять услугами, ценами, записями, отзывами, закрытыми датами и выгрузками.
- `admin` — помощник. Может смотреть записи, работать с клиентами, переносить и отменять записи, закрывать даты и модерировать отзывы.
- `viewer` — только просмотр записей и статистики.

Команды:

```text
/myid
/admins
/setrole 987654321 super_admin
/setrole 987654321 admin
/setrole 987654321 viewer
```

## Админ-панель

Команда `/admin` открывает панель мастера:

- сегодня, завтра, неделя и выбор даты;
- полная карточка записи с телефоном, Telegram ID, username, услугами, суммой и статусом;
- отмена, перенос и отметка «выполнено»;
- управление услугами и ценами;
- список клиентов, поиск и история записей клиента;
- закрытие дней и отдельных интервалов времени;
- модерация отзывов;
- CSV-выгрузка за день, неделю, месяц или по всем записям;
- статистика.

Обычный пользователь без роли не может открыть `/admin`.

## Данные

Проект по-прежнему хранит данные в JSON, чтобы бот было просто запускать без базы данных. Мастеру не нужно редактировать файлы вручную: услуги, цены, записи, закрытые даты и выгрузки управляются через `/admin`.

При чтении старые записи нормализуются в новую структуру, поэтому старые данные с полями `time`, `service`, `client_name` и `client_phone` не ломают запуск.
